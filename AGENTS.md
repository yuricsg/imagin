# Repository Instructions

## Project Layout

- The dashboard application lives in `dashboard/`.
- The lead capture API lives in `backend/`.
- Before editing Next.js code, read `dashboard/AGENTS.md` and the relevant files under `dashboard/node_modules/next/dist/docs/` after installing dependencies.

## Working Rules

- Do not run git operations that affect the remote unless explicitly requested.
- Keep changes small, tested, and documented.
- For chatbot work, update `ARCHITECTURE.md` or a feature-specific document whenever behavior, data contracts, or integration details change.
